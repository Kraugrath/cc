Dockerfile (No requirements.txt)

This installs Flask directly inside the image.

FROM python:3.10-slim

WORKDIR /app

COPY . .

RUN pip install flask

CMD ["python", "app.py"]


---

Run Steps

Build:

docker build -t flask-app .

Run:

docker run -p 5000:5000 flask-app

Open in browser:

http://localhost:5000
